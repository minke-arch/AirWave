# 영화 예매 서비스 (MovieWave) 디자인 스타일 가이드

본 문서는 **무비웨이브 (MovieWave)** 1차 MVP에 적용될 화면 레이아웃, 컬러 시스템, 타이포그래피 및 UI 구성 가이드라인을 정의합니다. 레퍼런스 모바일 웹 스크린샷 분석 결과를 바탕으로 수립되었습니다.

---

## 1. 디자인 아이덴티티 및 기본 테마

* **기본 테마**: **라이트 테마 (Light Theme) 기반**
  * 모바일 앱/웹 브라우저의 기본 배경은 깨끗하고 깔끔한 화이트와 밝은 그레이 톤으로 구성합니다.
* **부분 다크 블록 (Localized Dark Blocks)**:
  * 극장 내부의 어두운 분위기와 예매의 몰입도를 높이기 위해, **시간표 선택 후의 예매/좌석 배치도 영역(미니맵 포함) 및 큰 좌석 선택 창**만 배경을 어두운 차콜 그레이로 대비되게 설계합니다.

---

## 2. 브랜드 컬러 팔레트 (Color Palette)

### 2.1 메인 배경 & 텍스트 (Base & Background)
* **Body Background**: `#FFFFFF` (기본 흰색)
* **Section Border & Light Background**: `#F3F4F6` (연한 그레이), `#F9FAFB` (매우 연한 그레이)
* **Dark Section Background**: `#171717` (상영관 스크린 & 좌석 영역만 적용할 차콜 블랙)
* **Text Main**: `#111827` (짙은 차콜/검은색)
* **Text Secondary**: `#6B7280` (중간 회색)

### 2.2 시그니처 및 액션 컬러 (Primary & Accent)
* **CGV 시그니처 레드**: `#E71A0F` (GNB 로고, 중요 버튼, 활성화 테두리)
* **액티브 포인트 선홍색**: `#FF2E2E` (결제 및 최종 예매하기 활성화 상태)
* **그라데이션 포인트**: `linear-gradient(to right, #FF5E3A, #FF2E2E)` (하단 플로팅 예매 버튼용 오렌지-레드 그라데이션)

### 2.3 좌석 등급 및 상태 구분 (Seats Color Coding)
* **일반석 (Standard)**: `#FFFFFF` 배경 + 얇은 그레이 테두리
* **장애인석 (Handicapped)**: `#38BDF8` (하늘색 배경)
* **선택된 좌석 (Selected)**: `#FF2E2E` (레드 배경/테두리)
* **예매 완료 / 선택 불가 (Occupied / Disabled)**: `#E5E7EB` (연한 회색 배경) 또는 대각선 빗금 패턴 (`background-image`)

## 3. 타이포그래피 및 외부 폰트 (Typography)

* **전역 폰트**: **Noto Sans KR** (또는 Pretendard)
  * 영문뿐만 아니라 한글 가독성이 미려하게 떨어지도록 모던 샌드고딕 스타일을 유지합니다.
* **Next.js 구글 폰트 적용 설정**:
  * `app/layout.tsx`에서 Noto Sans KR을 아래와 같이 로드하여 tailwind 변수로 맵핑합니다.
  ```typescript
  import { Noto_Sans_KR } from 'next/font/google';

  const notoSansKr = Noto_Sans_KR({
    subsets: ['latin'],
    weight: ['400', '500', '700', '900'],
    variable: '--font-noto-sans',
  });
  ```

## 4. 주요 UI 컴포넌트 디자인 가이드

### 4.1 상단 네비게이션 & 글로벌 하단 바
* **GNB (Global Navigation Bar)**:
  * 모바일 화면 최상단 고정. 좌측 CGV 로고(Red), 우측 티켓/알림/검색 아이콘 배치.
* **하단 탭바**:
  * 아이콘 하단에 텍스트가 들어가는 형태. 활성화된 탭은 검은색 또는 포인트 컬러로 강조, 비활성화 탭은 연한 회색. 중앙의 '예매·예약'은 오렌지 그라데이션 원형 버튼으로 시선 유도.

### 4.2 영화 카드 및 캐러셀
* 영화 이미지 포스터 카드는 `aspect-[2/3]` 비율로 미려하게 정렬하며, 모서리는 둥글게(`rounded-xl` 또는 `rounded-2xl`) 처리합니다.
* 마우스 호버 또는 활성화 시 카드 스케일 업(`scale-105`) 및 그림자 효과로 입체감을 줍니다.

### 4.3 예매 시간표 & 날짜 탭
* 날짜 선택 탭: 선택된 날짜는 빨간색 원형 배경 또는 둥근 테두리에 흰색 글씨로 하이라이트 처리합니다. 토요일은 파란색(`text-blue-500`), 일요일은 빨간색(`text-red-500`) 폰트 색상을 적용합니다.
* 시간표 그리드: 상영 시작/종료 시간, 남은 좌석수(파란색 폰트), 상영관(예: 5관 2D) 정보가 간결하게 들어간 카드 형태.

### 4.4 결제 완료 티켓 디자인
* 마이페이지/티켓창에 표시될 모바일 티켓은 실제 티켓처럼 상단과 하단이 절취선(점선)으로 나누어지는 카드 디자인 적용.
* 하단에 모의 바코드 및 예매번호, 관람 시간, 좌석번호(`L6`, `L7`)를 크게 배치합니다.
